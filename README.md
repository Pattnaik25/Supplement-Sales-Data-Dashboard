# Supplement-Sales-Data-Dashboard

Dataset link - https://www.kaggle.com/datasets/zahidmughal2343/supplement-sales-data
The dataset used in this project, captures weekly sales data for a range of health and wellness supplements from January 2020 to April 2025. It includes products across various categories such as Protein, Vitamins, Omega, and Amino Acids. Sales data is collected from multiple e-commerce platforms, including Amazon, Walmart, and iHerb, and spans several geographic markets, including the USA, UK, and Canada.
